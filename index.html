# Black Tie Affair — Style Selector An interactive tuxedo style picker built for Black Tie Affair, a luxury formalwear rental business in Honolulu, Hawaiʻi. ## What it does A 5-step style quiz that lets customers choose their tuxedo preferences before completing their rental order: 1. **Fit** — Classic or Slim 2. **Lapel** — Notch, Peak, or Shawl 3. **Color** — Classic Black, Midnight Blue, or White Jacket 4. **Neckwear** — Bow Tie or Neck Tie 5. **Waist** — Vest or Cummerbund Sash At the end, customers copy their selections and continue to the order form. ## Setup 1. Open `index.html` 2. Find this line near the top of the script: ``` const TALLY_FORM_URL = 'https://tally.so/r/YOUR_FORM_ID'; ``` 3. Replace `YOUR_FORM_ID` with your actual Tally form ID 4. Deploy to Vercel ## Embedding in Tally 1. Deploy this project to Vercel 2. Copy the live Vercel URL 3. In your Tally form, type `/` → **Embed anything** → paste the URL ## Built with - Vanilla HTML, CSS, JavaScript - Dancing Script + Lato (Google Fonts) - No dependencies or build step required ## Powered by [Kalamaku AI](https://kalamakuai.com) — AI Automation & Strategy Studio, Honolulu, Hawaiʻi
